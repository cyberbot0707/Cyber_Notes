# CryptoNotes — Защищенный цифровой сейф заметок

Веб-приложение на Python и Flask, разработанное для безопасного хранения личных текстовых записей и файлов с распределением по категориям.

##  Технологический стек
* **Backend:** Python 3, Flask
* **Database & ORM:** SQLite, SQLAlchemy
* **Frontend:** HTML

## Реализованные механизмы безопасности (OWASP)
1. **Хеширование паролей:** Использование криптографического алгоритма PBKDF2 с уникальной солью для каждого пользователя (библиотека Werkzeug). Пароли не хранятся в открытом виде.
2. **Защита от IDOR (Insecure Direct Object Reference):** Строгая проверка владельца сессии при просмотре, редактировании или удалении записей. Пользователи полностью изолированы друг от друга.
3. **Защита от SQL-инъекций:** Безопасное параметризированное взаимодействие с СУБД через SQLAlchemy ORM без использования прямых уязвимых SQL-запросов.
4. **Безопасная загрузка файлов:** Валидация расширений файлов по белому списку, ограничение максимального размера входящего контента и санитария имен через `secure_filename` для защиты от уязвимости Path Traversal.
5. **Защита данных (Privacy):** Применение CSS-эффектов (Blur) для маскирования конфиденциальной информации на экране от плечевого подглядывания.
